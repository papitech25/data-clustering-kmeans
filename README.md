# data-clustering-kmeans
K-Means clustering implementation with data visualization using Python.

## Description
This project demonstrates unsupervised learning using the K-Means clustering algorithm.

Synthetic data is generated and grouped into clusters automatically. The results are visualized to show how the algorithm identifies patterns in data.

## Features
- Data generation
- K-Means clustering
- Visualization of clusters and centroids

## Technologies
- Python
- NumPy
- Matplotlib
- Scikit-learn
